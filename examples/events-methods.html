Events Or Methods

Adding events to the element is easier just by addding an attribute "gamm-events" with a JSON data starts with the key as the name of the event and value as the function inside "events" parameters in the library class. 

			
				HTML
	<div id="main-content">		
		<button gamm-events="{'click':'show_alert'}" >Click Me For Alert!</button>
	</div>

Here you can see we add a JSON format data inside "gamm-events" with the key as "click" and the value is the function name "show_alert". 

			
				Javascript
	new Gamm({
		element : "#main-content",
		events : {
			show_alert : function(){

				alert("Hello Gamm.js");

			}
		}
	});
